O que é o ANgularJS

Um framework para criar aplicações web, de maneira rápida, testável e de fácil manutenção.

Princípios do AngularJS

  • Desenvolvimento Rápido
  • Escrever menos código
  • Modularidade
  • Pensado para ser testado

O que o AngularJS oferece

  • Controllers
  • Templates e Data bindings
  • Services
  • Directives
  • Integração com REST

O que o AngularJS oferece parte 2

  • Injeção de dependencias
  • Transições (em CSS3)
  • Filtros
